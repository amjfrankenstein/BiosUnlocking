L51 System Firmware (BIOS) Family - Revision History

Models supported

  • HP Z230 Tower Workstation
  • HP Z230 SFF Workstation
  • HP Z228 Workstation

History

Version 1.63

Updates Intel CPU Microcode to 0x28.

Includes enhancements to mitigate security vulnerabilities CVE-2020-0543, CVE-2020-0548, CVE-2020-0549, CVE-2020-0528 and CVE-2020-0529.

HP strongly recommends promptly transitioning to this updated BIOS version.

Version 1.62

Updates Intel CPU Microcode to 0x27.

Includes enhancements to mitigate security vulnerabilities CVE-2018-12126, CVE-2018-12127, CVE-2018-12130 and CVE-2019-11091.

Fixes an issue where the boot HDD is incorrectly reported as “other” in EFI boot order reported by BiosConfigUtility and Replicated Setup.

Version 1.61

Updates Intel CPU Microcode to 0x25.

Includes enhancement to mitigate CVE-2018-3639 and CVE-2018-3640.

Fixes an issue where PCR2 values could change causing BitLocker to enter recovery mode.

Version 1.59

Updates Intel CPU Microcode to 0x24.

Includes enhancement to mitigate CVE-2017-5715. HP strongly recommends promptly transitioning to this updated BIOS version.

Version 1.58

Reverts Intel processor microcode to 0x1D.

Updates Intel LOM UEFI driver to v0.0.18.

Fixes an issue where EFI PXE boot would fail after reboot with AMT disabled.

Fix the Fort Knox issue 17B4: Wake-on LAN bypasses Power-on Password arbitrarily.

Version 1.57

Update the microcode to M32306C3_00000023

Fix the changes of some UEFI variable formats after v1.55 cause abnormal booting behaviors.

Fix an issue that TPM is not found in Windows, when set TPM is "Hidden" in F10 then changes TPM state to "Available" via BCU in Windows.

Fix the Fort Knox issue 17B4: Wake-on LAN bypasses Power-on Password arbitrarily.

Version 1.55

This release includes updates to address “Microsoft Secure Boot Security Feature Bypass” as described in CVE-2016-3287 and CVE-2016-3320. HP strongly recommends promptly transitioning to this updated BIOS version which supersedes all previous releases.

Updates UEFI Network Stack to resolve issue when attempting to PXE boot with Secure Boot enabled.

Version 1.54

WARNING: After installing this BIOS version onto the system, prior BIOS versions cannot be installed onto the system.

This release improves security of UEFI code and variables and includes mitigation for the following vulnerabilities.

HP strongly recommends promptly transitioning to these updated BIOS versions which supersede all previous releases.

Intel-TA DOC #548744

VU#631788

VU#766164

VU#976132 

Version 1.53

Updates Intel GOP driver to v5.5.1033.

Updates Intel vBIOS to v5.5.1039.

Updates Legacy PXE Option ROM to v1.5.56.

Fixes an issue where a Thunderbolt card is identified incorrectly in F10 Setup.

Fixes an issue where system would not be assigned IP address during PXE boot.

Fixes an issue where KVM sessions would not display graphics output, then close shortly after boot.

Fixes an issue where power loss during MEFW update may make ME permanently unavailable.

Version 1.52

Adds support for Microsoft Windows 10.

Adds the ability to disable the Option ROM for the embedded NIC.

Fixes an issue where system could not access WebBIOS utility with certain RAID controller cards.

Fixes an issue where user would receive an error message when attempting to read fan speeds via WMI (Z230 SFF only).

Version 1.42

WARNING: After installing this BIOS version onto the system, prior BIOS versions cannot be installed onto the system.

Fixes a security vulnerability where UEFI variables can be overwritten which could potentially lead to bypass of security features or denial of service. HP strongly recommends transitioning promptly to this updated BIOS version.

Update Haswell C0 CPU microcode to 0x1C.

Update Intel GOP driver to 5.5.1027

Update Intel vBIOS to 5.5.1031

Enables Secure Erase support with Z Turbo Drive PCIe-based SSD storage solution.

Adds support for future Z Turbo Drive versions.

Fixes an issue which resulted in "fan not detected" error after AC power cycle.

Fixes an issue where POST screen was blurred with certain graphics cards.

Fixes an issue where BitLocker would issue an error message stating that the system configuration changed every boot.

Fixes an issue where system would hang during large file transfers with AMT enabled.

Version 1.40

WARNING: After installing this BIOS version onto the system, prior BIOS versions cannot be installed onto the system.

Provides improved security for UEFI EDK2. HP strongly recommends transitioning promptly to this updated BIOS version.

Version 1.33

Updates Intel processor uCode to 0x1A.

Updates Intel MEBX to v9.0.0.0029.

Updates Intel GOP driver to v5.0.1037..

Updates Intel VBIOS to v1028.

Increases amount of available system memory when Thunderbolt card is not installed.

Modifies memory speed information to report actual speed rather than SPD information.

Fixes an issue where the "Update Address" setting was missing from WMI.

Fixes an issue where the HP Z Turbo Drive (PCIe-based SSD storage solution) and some graphics cards were shown in system tray as removable devices.

Fixes an issue where the HP Z Turbo Drive with DriveLock enabled would not work after S3 resume.

Fixes an issue where 15-in-1 Media Card Reader info was not shown in F10 menu.

Fixes an issue where Happauge TV tuner PCIe cards were not detected by the system.

Fixes an issue where Teradici TERA2220 PCoIP host card would hang after reboot.

Fixes an issue where DisplayPort port D would not work with certain processors (Z230 SFF only).

Fixes an issue where system would hang at POST (no display) with certain combinations of DisplayPort and DVI monitors.

Fixes an issue where the system would hang after booting from Optical Drive with IDE mode.

Fixes an issue where broadcasting ARP replies caused a large amount of unnecessary network traffic.

Disables PCIe root port Latency Tolerance Reporting (LTR) to eliminate blue screen failure after hibernation.
